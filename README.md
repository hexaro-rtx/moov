# Xray Server pour Tunnel Mobile

Configuration V2Ray (VMess + WebSocket) déployable sur Railway.

## Déploiement
1. Fork ce repo
2. Connecte Railway à ton GitHub
3. Déploie automatiquement

## Configuration Client (NPV Tunnel)
- Protocol : VMess
- Network : WebSocket
- Path : /graphql
- TLS : Activé
- SNI : [ton-url.railway.app]
